# BTC Price Tracker

A real-time Bitcoin price tracker built with Next.js 14, NextAuth.js, and Tailwind CSS.

## Features

- 📈 Live BTC/USD price from CoinGecko (auto-refresh every 30s)
- 🔐 Google Sign In via NextAuth.js
- 🌑 Modern dark UI with orange Bitcoin accents
- ⚡ Next.js 14 App Router

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Auth:** NextAuth.js v4 with Google provider
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Data:** CoinGecko public API

## Deployment

Deployed on [Render](https://render.com) at https://builder-bot-ab6k.onrender.com
